Nmap
Network Mapper — Découverte d'hôtes, scan de ports, fingerprinting de services et OS
Réseau Scanner v7.94+
⚠️
Utilisez Nmap uniquement sur des réseaux et systèmes dont vous êtes propriétaire ou pour lesquels vous disposez d'une autorisation écrite. Les scans non autorisés sont illégaux.

📦 Installation

Installation
# Debian / Ubuntu / Kali
sudo apt install nmap

# Arch Linux
sudo pacman -S nmap

# macOS (Homebrew)
brew install nmap

# Vérifier la version
nmap --version

⚙️ Générateur de commande

nmap -sS 192.168.1.1

📋 Cheatsheet complet

Scans de base

Scans essentiels
# Scan de base (1000 ports TCP)
nmap 192.168.1.1

# Scan SYN stealth (root) — le plus courant
nmap -sS 192.168.1.1

# Scan de tout le sous-réseau
nmap -sn 192.168.1.0/24

# Scan des 65535 ports
nmap -p- 192.168.1.1

# Scan rapide avec version + OS
nmap -sS -sV -O -T4 192.168.1.1

# Full agressif
nmap -A -T4 -p- 192.168.1.1

# Sans résolution DNS (plus rapide)
nmap -n -Pn 192.168.1.1

# Scan de plusieurs cibles
nmap 192.168.1.1 192.168.1.2 192.168.1.5
nmap 192.168.1.1-50

Détection de services et OS

Version & OS
# Détection de version de service
nmap -sV 192.168.1.1

# Intensité de détection (0-9)
nmap -sV --version-intensity 9 192.168.1.1

# Détection d'OS
nmap -O 192.168.1.1

# Forcer la détection OS
nmap -O --osscan-guess 192.168.1.1

Scripts NSE (Nmap Scripting Engine)

NSE Scripts
# Scripts par défaut
nmap -sC 192.168.1.1

# Scripts par catégorie
nmap --script=vuln 192.168.1.1
nmap --script=auth 192.168.1.1
nmap --script=discovery 192.168.1.1
nmap --script=safe 192.168.1.1
nmap --script=exploit 192.168.1.1

# Scripts HTTP
nmap --script=http-title,http-headers,http-methods -p 80,443 192.168.1.1
nmap --script=http-enum 192.168.1.1

# Brute force via scripts
nmap --script=ssh-brute -p 22 192.168.1.1
nmap --script=ftp-brute -p 21 192.168.1.1

# Vulnérabilités EternalBlue (MS17-010)
nmap --script=smb-vuln-ms17-010 -p 445 192.168.1.1

# Enumération SMB
nmap --script=smb-enum-shares,smb-enum-users -p 445 192.168.1.1

# Lister les scripts disponibles
ls /usr/share/nmap/scripts/ | grep ssh

Évasion de firewall & IDS

Firewall Evasion
# Fragmentation des paquets
nmap -f 192.168.1.1

# Leurres (decoys) pour masquer l'IP réelle
nmap -D RND:10 192.168.1.1
nmap -D 10.0.0.1,10.0.0.2,ME 192.168.1.1

# Source port spoofing
nmap --source-port 53 192.168.1.1

# Délai entre paquets
nmap --scan-delay 1s 192.168.1.1

# Via proxy SOCKS
nmap --proxies socks4://127.0.0.1:9050 192.168.1.1

# Taille de MTU personnalisée
nmap --mtu 24 192.168.1.1

Sortie et rapports

Output formats
# Normal text
nmap -oN scan.txt 192.168.1.1

# XML (pour Metasploit, etc.)
nmap -oX scan.xml 192.168.1.1

# Grepable format
nmap -oG scan.gnmap 192.168.1.1

# Tous les formats en même temps
nmap -oA scan_results 192.168.1.1

# Reprendre un scan interrompu
nmap --resume scan.gnmap
