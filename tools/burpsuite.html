📋 Modules principaux

Proxy & Intercept

Proxy — Utilisation
Raccourcis clavier essentiels :
Ctrl+F      — Forward (laisser passer la requête)
Ctrl+A      — Send to all tabs
Ctrl+R      — Send to Repeater
Ctrl+I      — Send to Intruder
Ctrl+Shift+R — Send to Scanner

Actions sur les requêtes interceptées :
- Clic droit → "Send to Repeater" → modifier et rejouer
- Clic droit → "Send to Intruder" → fuzzing/brute force
- Clic droit → "Do intercept" → désactiver pour cette requête
- Clic droit → "Save item" → sauvegarder
- Clic droit → "Copy as curl command" → exporter

Repeater

Repeater — Tests manuels
# Le Repeater permet de modifier et rejouer des requêtes manuellement
# Utile pour : SQLi manuelle, tests XSS, exploitation logique métier

Workflow :
1. Intercepter une requête (Proxy → Intercept ON)
2. Clic droit → Send to Repeater (Ctrl+R)
3. Modifier la requête dans le Repeater
4. Cliquer "Send" pour envoyer
5. Analyser la réponse

Tests courants dans le Repeater :
# Injection SQL basique
GET /page?id=1' OR '1'='1

# XSS
GET /search?q=<script>alert(1)</script>

# Path traversal
GET /download?file=../../../../etc/passwd

# SSRF
POST /fetch  body: url=http://169.254.169.254/latest/meta-data/

Intruder — Fuzzing & Brute Force

Intruder
Types d'attaques Intruder :
Sniper      — 1 payload, 1 position (le plus courant)
Battering Ram — 1 payload, toutes les positions simultanément
Pitchfork  — N payloads pour N positions (username:password)
Cluster Bomb — Toutes les combinaisons (brute force)

Workflow brute force login :
1. Intercepter une requête POST de login
2. Send to Intruder
3. Onglet "Positions" → Clear § → sélectionner le champ password → Add §
4. Onglet "Payloads" → Type: Simple list → charger wordlist
5. Start Attack
6. Analyser la longueur de réponse (différente = succès)

Types de payloads utiles :
Simple list   — Wordlist statique
Runtime file  — Charger un fichier en temps réel
Numbers       — Séquences numériques
Brute forcer  — Caractères + longueur
Username generator — Variations d'un nom

Utilisations courantes :
