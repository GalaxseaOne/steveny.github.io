⚡ Commandes principales

Attaques courantes
# Attaque dictionnaire (-a 0)
hashcat -m 0 -a 0 hash.txt /usr/share/wordlists/rockyou.txt

# NTLM avec dictionnaire
hashcat -m 1000 -a 0 ntlm_hashes.txt rockyou.txt

# NTLMv2 (Responder)
hashcat -m 5600 -a 0 ntlmv2.txt rockyou.txt

# Kerberoast
hashcat -m 13100 -a 0 kerberoast.txt rockyou.txt

# AS-REP Roast
hashcat -m 18200 -a 0 asrep.txt rockyou.txt

# WPA handshake (après capture)
hashcat -m 22000 -a 0 capture.hc22000 rockyou.txt

# Avec règles
hashcat -m 0 -a 0 hash.txt rockyou.txt -r /usr/share/hashcat/rules/best64.rule
hashcat -m 0 -a 0 hash.txt rockyou.txt -r /usr/share/hashcat/rules/d3ad0ne.rule

# Afficher les mots de passe crackés
hashcat -m 0 --show hash.txt

Brute force par masque (-a 3)

Masques Hashcat
Caractères de masque :
?l  — minuscules (a-z)
?u  — majuscules (A-Z)
?d  — chiffres (0-9)
?s  — caractères spéciaux
?a  — tous (?l?u?d?s)
?b  — bytes 0x00-0xff

# 8 minuscules
hashcat -m 0 -a 3 hash.txt "?l?l?l?l?l?l?l?l"

# Mot de passe de 6 à 8 caractères (increment)
hashcat -m 0 -a 3 hash.txt "?a?a?a?a?a?a" --increment --increment-min 6

# Pattern commun : Mot+chiffres (ex: password123)
hashcat -m 0 -a 3 hash.txt "?l?l?l?l?l?l?l?d?d?d"

# Majuscule + 6 min + 2 chiffres
hashcat -m 0 -a 3 hash.txt "?u?l?l?l?l?l?l?d?d"

# Fichier de masques prédéfinis
hashcat -m 0 -a 3 hash.txt /usr/share/hashcat/masks/rockyou-7-2592000.hcmask

Règles de transformation

Règles hashcat (transformations)
# Règles incluses dans Kali
ls /usr/share/hashcat/rules/
best64.rule         — 64 meilleures règles
d3ad0ne.rule       — ~34k règles
dive.rule           — ~99k règles (lent)
rockyou-30000.rule — basé sur rockyou
KoreLogicRules*    — règles professionnelles

# Générer des règles avec combinator
hashcat -m 0 -a 0 hash.txt wordlist.txt -r best64.rule -r d3ad0ne.rule
