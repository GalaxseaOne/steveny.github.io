Google Dorks
Opérateurs de recherche avancés pour l'OSINT et la découverte d'informations sensibles
OSINT Recon GHDB
ℹ️
Les Google Dorks exploitent les opérateurs de recherche avancés de Google pour trouver des informations publiquement accessibles. Tout ce qui est indexé est légalement accessible en lecture.

⚙️ Générateur de Dork

site:exemple.com

📖 Opérateurs de base

Syntaxe des opérateurs
Opérateur          Description
site:domaine.com   Limiter la recherche à un domaine
inurl:mot          Mot dans l'URL
intitle:mot        Mot dans le titre de la page
intext:mot         Mot dans le contenu
filetype:pdf       Type de fichier spécifique
ext:sql            Extension de fichier
cache:url          Version en cache
link:url           Pages qui pointent vers cette URL
related:url        Sites similaires
"texte exact"       Recherche de phrase exacte
-mot               Exclure un mot
mot1 OR mot2       L'un ou l'autre
*                  Wildcard
